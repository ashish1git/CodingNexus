# Welcome to Web Dev Bootcamp

Welcome to the **Coding Nexus Club Bootcamp**! This comprehensive 3-day course will transform you into a confident full-stack developer using the **MERN stack** (MongoDB, Express, React, Node.js).

## What You'll Learn

### Day 1: Backend Fundamentals
Learn to build a robust backend server with Node.js and Express. Connect your server to MongoDB and implement CRUD operations.

## Prerequisites

Before you start, you'll need:
- A computer with Windows, macOS, or Linux
- Basic JavaScript knowledge
- A text editor (VS Code is recommended)
- Internet connection for MongoDB Atlas setup

## Getting Started

Begin your journey with **[Day 1: Backend Fundamentals](./curriculum/day-1-backend-fundamentals.md)**.

## Course Structure

This bootcamp is organized by day with detailed lessons, code examples, and hands-on exercises. Each section builds upon the previous one to create a complete understanding of full-stack development.
